RunG 개인정보처리방침

시행일: 2025년 7월 18일

RunG 앱을 사용해 주셔서 감사합니다. 저희는 귀하의 개인정보와 데이터를 보호하는 데 최선을 다하고 있습니다. 이 정책은 수집, 사용 및 보호 방법을 설명합니다.

1. 수집하는 정보

  • 계정 정보 (Apple ID 또는 이메일)
  • 러닝 데이터 (경로, 페이스, 시간, 거리, 케이던스 등)
  • 기기 정보 (운영체제 버전, 기기 모델, 언어 설정)
  • 날씨 데이터 (위치 기반 표시 목적)

2. 데이터 사용 방법

  • 러닝 세션 표시 및 분석
  • 위치 기반 지역 날씨 정보 제공
  • 앱 기능 및 사용자 경험 개선

개인 정보를 제3자에게 판매하거나 공유하지 않습니다.

3. 필요한 권한

  • 위치 접근 권한: 경로 기록 및 날씨 정보 수집용
  • 네트워크 접근 권한: 데이터 동기화(사용자 요청 시) 및 날씨 정보 가져오기

4. 데이터 보안

로컬 저장소 및 Supabase 클라우드 서비스를 이용하여 데이터를 보호하며, 명시적 동의 없이는 접근하거나 공유하지 않습니다.

5. 귀하의 권리

  • 언제든지 러닝 데이터를 조회, 편집 또는 삭제할 수 있습니다.
  • 설정에서 계정 초기화 또는 앱 데이터 삭제 가능
  • 전체 계정 삭제는 문의해 주시기 바랍니다.
